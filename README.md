# Open Grove Labs

Open source tools for modern families — thoughtfully designed, privacy-first, and free forever.

## About

Open Grove Labs builds practical apps for parents and playful tools for kids. Every project is crafted with care, designed to be simple, reliable, and delightful to use. No ads, no data harvesting, no dark patterns — just useful software for everyday family life.

## Featured Projects

### Basil
Simple, open-source budgeting for busy families. Take control of spending, track goals, and build healthier financial habits.

### Smashy Keys
A colorful keyboard-tapping playground for toddlers. Made for laughter, learning, and safe screen time.

### Early Reader (Working Title)
Interactive activities that help young readers build confidence, one step at a time.

## Development

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/open-grove-labs.git
cd open-grove-labs
```

2. Install dependencies:
```bash
yarn install
```

3. Start the development server:
```bash
yarn dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
# or
yarn build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
# or
yarn preview
```
